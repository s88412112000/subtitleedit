# 自動翻譯 (Auto Translate)

使用各種翻譯引擎和 AI 服務自動翻譯字幕。

- **選單:** 自動翻譯 (Auto translate) → 自動翻譯 (Auto translate)...
- **快捷鍵:** 可設定

<!-- Screenshot: Auto translate window -->
![Auto Translate](../screenshots/auto-translate.png)

## 如何使用

1. 開啟 **自動翻譯 (Auto translate) → 自動翻譯 (Auto translate)...**
2. 選擇一個翻譯引擎
3. 選擇來源語言和目標語言
4. 點擊 **翻譯 (Translate)** 開始
5. 在網格中審核翻譯結果
6. 點擊 **確定 (OK)** 套用

## 支援的引擎

- **Google Translate (V1)** — 免費的 Google 翻譯
- **Google Translate (V2)** — Google Cloud 翻譯 (需要 API 金鑰)
- **Microsoft Translator** — Azure Cognitive Services (需要 API 金鑰)
- **DeepL** — DeepL 翻譯 (需要 API 金鑰)
- **Libre Translate** — 開源、可自行架設的翻譯
- **MyMemory** — 免費翻譯記憶庫
- **Ollama** — 本機基於 LLM 的翻譯
- **Anthropic Claude** — AI 翻譯 (需要 API 金鑰)
- **OpenAI ChatGPT** — AI 翻譯 (需要 API 金鑰)
- **Groq** — AI 翻譯 (需要 API 金鑰)
- **LM Studio** — 本機 LLM 翻譯

## 引擎設定

根據所選的引擎，您可能需要提供：

- **API 金鑰 (API Key)** — 雲端服務的驗證金鑰
- **API 網址 (API URL)** — 自訂端點網址 (適用於自行架設的服務)
- **模型 (Model)** — 要使用的特定模型 (適用於 AI 引擎)

## 翻譯審核

翻譯網格會將原始文字與翻譯後的文字並排顯示。您可以在接受之前個別編輯翻譯內容。

## 鍵盤快捷鍵

| 按鍵 | 動作 |
|-----|--------|
| Escape | 關閉 / 取消 |
| F1 | 開啟說明 |
