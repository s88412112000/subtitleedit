# 空白影片 (Blank Video)

生成一個空白影片檔案，用於測試或預覽字幕。

- **選單:** 影片 (Video) → 生成空白影片 (Generate blank video)...
- **快捷鍵:** 可設定

<!-- Screenshot: Blank video window -->
![Blank Video](../screenshots/blank-video.png)

## 如何使用

1. 開啟 **影片 (Video) → 生成空白影片 (Generate blank video)...**
2. 設定影片解析度 (寬度 × 高度)
3. 選擇影格率 (frame rate)
4. 設定持續時間 (分鐘)
5. 選擇背景類型
6. 點擊 **生成 (Generate)**

## 背景選項

- **棋盤格圖片 (Checked image)** — 經典的棋盤格圖案 (適合用於透明度測試)
- **單色 (Solid color)** — 單一純色背景
- **背景圖片 (Background image)** — 使用自訂圖片檔案作為背景

## 額外選項

- **生成時間碼 (Generate time codes)** — 將時間碼顯示燒錄到影片中
- **使用來源解析度 (Use source resolution)** — 符合現有影片的解析度

## 批次模式

可以將多個空白影片加入工作佇列以進行批次生成。

## 鍵盤快捷鍵

| 按鍵 | 動作 |
|-----|--------|
| Escape | 關閉 / 取消 |
| F1 | 開啟說明 |
